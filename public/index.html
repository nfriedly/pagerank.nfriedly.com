    About Page Rank

    PageRank, named after Larry Page, is one of several metrics Google uses to determine search result rankings.

    It is an indication of how important Google believes a page is, based on how many other web pages link to it and how "important" those web pages are. (What PageRank the linking pages have.)

    Other metrics that search engines use in determining results include the load time of the website, the validity of the markup, and above all, the content of the website.
